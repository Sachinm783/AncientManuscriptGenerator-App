📜 Ancient Manuscript Generation – Android App
🚀 Overview

This project is an Android-based application designed to generate synthetic ancient manuscript images using Generative AI techniques. It focuses on supporting low-resource OCR research by creating realistic handwritten-style manuscript data.

The app enables users to input text and generate manuscript-style outputs that mimic ancient scripts, which can be used for:

OCR model training

Dataset augmentation

Historical document simulation

🎯 Features

📝 Text-to-Manuscript Generation

🤖 AI-powered synthetic script creation

📱 User-friendly Android interface

🖼️ Realistic manuscript-style rendering

💾 Save and export generated images

⚡ Lightweight and efficient processing

🧠 Research Motivation

This project is part of the research:

“Generative AI for Synthetic Document Creation for Low-Resource OCR”

Low-resource languages and ancient scripts often lack sufficient datasets. This app helps address that gap by generating synthetic training data for improving OCR systems.
